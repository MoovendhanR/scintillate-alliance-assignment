## Run Locally

```bash
  git clone https://github.com/MoovendhanR/scintillate-alliance-assignment
```

Go to the project directory

```bash
  cd scintillate-alliance-assignment
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run start
```
